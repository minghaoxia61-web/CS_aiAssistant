# 动态规划

动态规划（Dynamic Programming, DP）是求解最优化问题的一种重要算法设计方法，由 Richard Bellman 提出。它通过将复杂问题分解为相对简单的子问题，并保存子问题的解以避免重复计算，从而大幅提升效率。掌握动态规划是算法学习的分水岭，也是各类笔试、面试中的核心考点。

## 概念定义

- **最优子结构（Optimal Substructure）**：问题的最优解包含其子问题的最优解。
- **重叠子问题（Overlapping Subproblems）**：递归求解时同一子问题被多次重复计算。
- **状态（State）**：描述子问题的参数集合，如 `dp[i]`、`dp[i][j]`。
- **状态转移方程（State Transition Equation）**：表达状态间递推关系的公式。
- **无后效性**：当前状态一旦确定，未来决策只与当前状态有关，而与到达该状态的路径无关。

## 核心要点

### 动态规划的设计步骤

1. **刻画最优解的结构特征**，确认具备最优子结构。
2. **定义状态**：明确 `dp[i]` 或 `dp[i][j]` 的含义。
3. **写出状态转移方程**：核心，决定算法正确性。
4. **确定边界条件**（初始值）与计算顺序（自底向上或自顶向下记忆化）。
5. **（可选）记录决策**，回溯还原最优解方案。

### 适用问题类型

- 最优化问题（最大值 / 最小值 / 最长 / 最短）。
- 计数问题（方案数）。
- 求存在性问题（能否达到某状态）。

### 经典问题

#### 1. 0-1 背包问题

有 n 件物品，第 i 件重量 `w[i]`、价值 `v[i]`，背包容量 W，每件物品最多取一次，求最大价值。

状态：`dp[i][j]` 表示前 i 件物品、容量为 j 时的最大价值。
转移：

```
dp[i][j] = dp[i-1][j],                            不选第 i 件
dp[i][j] = max(dp[i][j], dp[i-1][j-w[i]] + v[i]), 选第 i 件（j >= w[i]）
```

空间优化为一维：`dp[j] = max(dp[j], dp[j-w[i]] + v[i])`，注意 j 需逆序遍历。

#### 2. 最长公共子序列（LCS）

字符串 X、Y 的最长公共子序列长度。`dp[i][j]` 表示 X 前 i 字符与 Y 前 j 字符的 LCS 长度。

```
若 X[i]==Y[j]: dp[i][j] = dp[i-1][j-1] + 1
否则: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

#### 3. 编辑距离

将字符串 A 转换为 B 所需的最少操作次数（插入 / 删除 / 替换）。

```
若 A[i]==B[j]: dp[i][j] = dp[i-1][j-1]
否则: dp[i][j] = 1 + min(dp[i-1][j],      删除
                         dp[i][j-1],      插入
                         dp[i-1][j-1])    替换
```

## 代码示例

### 0-1 背包（C，一维优化）

```c
int knapsack(int w[], int v[], int n, int W) {
    int dp[W + 1];
    for (int j = 0; j <= W; j++) dp[j] = 0;
    for (int i = 0; i < n; i++) {
        for (int j = W; j >= w[i]; j--) {  // 逆序！
            if (dp[j - w[i]] + v[i] > dp[j])
                dp[j] = dp[j - w[i]] + v[i];
        }
    }
    return dp[W];
}
```

### 最长公共子序列（Python）

```python
def lcs(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i - 1] == s2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    return dp[m][n]
```

### 编辑距离（Python）

```python
def edit_distance(a, b):
    m, n = len(a), len(b)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1): dp[i][0] = i
    for j in range(n + 1): dp[0][j] = j
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if a[i - 1] == b[j - 1]:
                dp[i][j] = dp[i - 1][j - 1]
            else:
                dp[i][j] = 1 + min(dp[i - 1][j],
                                   dp[i][j - 1],
                                   dp[i - 1][j - 1])
    return dp[m][n]
```

## 常见考点

1. **状态转移方程推导**：给出问题，要求写出 dp 数组定义、转移公式与边界条件。
2. **0-1 背包 vs 完全背包**：完全背包中物品可无限取，内层循环改为正序；常考二者差异与空间优化。
3. **空间优化**：二维压成一维时循环方向的判断，是高频陷阱题。
4. **LCS / LIS（最长递增子序列）**：LIS 可用二分优化到 O(n log n)。
5. **记忆化搜索 vs 递推**：自顶向下记忆化与自底向上递推的等价性，常作为代码改写题。
6. **路径计数 / 矩阵最小路径和**：典型二维 DP。
7. **无后效性判断**：分析某状态定义是否满足无后效性，是设计 DP 的关键。
8. **常见错误**：初始化错误、循环顺序错误、状态定义歧义等，需在练习中养成检查习惯。
